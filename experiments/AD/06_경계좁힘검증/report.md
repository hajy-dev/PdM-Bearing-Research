# 증명: 이상탐지 경계 좁힘 검증 + 임계값 확정

## 변경 내용
- ANOMALY_THRESHOLD를 0.4 → 0.6으로 변경 (정상 > 0.7은 유지)
- 정상/이상 간격: 0.3 → 0.1로 축소
- 초기 열화(HI 0.4~0.6)도 이상에 포함

## 검증 목적
HI 기반 분리에서 AUC 0.988이 정상/이상 간격이 넓어서(0.3) 쉬운 과제였던 것인지, 실제 모델 능력인지 검증

## 결과

| 모델 | ≤0.4 (간격0.3) | ≤0.6 (간격0.1) | 변화 |
|------|---------------|---------------|------|
| IsolationForest | Det 97.2% / AUC 0.988 | Det **80.2%** / AUC **0.970** | AUC -0.018 |
| Autoencoder | Det 93.9% / AUC 0.985 | Det **87.9%** / AUC **0.977** | AUC -0.008 |
| OneClassSVM | Det 96.0% / AUC 0.986 | Det **81.2%** / AUC **0.941** | AUC -0.045 |

## 판정
**모델 실력 확인.** 간격을 1/3로 줄여도 AUC 0.94~0.97 유지. 초기 열화도 80~88% 탐지 가능.

## 임계값 ≤0.6 확정 이유
1. 스토리텔링 시나리오에서 MEDIUM 단계(01:30) 감지가 핵심 — ≤0.4면 HIGH(02:00)까지 못 잡음
2. 이상탐지는 RUL 갱신의 트리거 — 조기 감지해야 알림 발송 가능
3. T분 지속 조건(노이즈 필터링)이 오탐 방지
4. 예지보전 = 조기 탐지가 목적
5. AUC 0.97, Det Rate 80%+로 실용 수준 검증 완료
