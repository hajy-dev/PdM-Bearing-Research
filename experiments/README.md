## 📁 구조

각 실험 폴더는 동일한 3개 하위로 구성:

```
{실험번호}_{설명}/
├── report.md       # 실험 보고서 (가설·결과·결론)
├── src/            # 실험에 사용된 코드 스냅샷
└── results/        # 지표·시각화 (raw 데이터는 .gitignore로 제외)
```
